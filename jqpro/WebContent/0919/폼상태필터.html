사용 불가능 / 가능 (변경)
실행시 동적으로 이벤트에 의해서 사용가능 또는 사용불가능 변경할 수 있다.
attr('속성명', '값') : attr("id", "aaa") / attr('type', 'button')
prop('속성명', 'boolean값') :
상태값을 설정 : disabled, selected, checked, multiple, readonly
prop('disabled', true) : 사용 불가능 설정
prop('disabled', false) : 사용 가능 설정
